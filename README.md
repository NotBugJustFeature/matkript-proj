## Téma

A projekt témája a **Carmichael-számok** vizsgálata. A feladat része Korselt kritériumának bizonyítása, annak belátása, hogy minden Carmichael-szám páratlan és legalább három különböző prímtényezője van, valamint egy Python-alapú keresőalgoritmus megvalósítása. A projekt kísérleti része a Carmichael-számok eloszlását vizsgálja `C(B)` számlálófüggvénnyel, log-log ábrával, az Erdős-féle heurisztikus becsléssel való összehasonlítással és a prímosztók számának hisztogramjával.
